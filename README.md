# SureShift

### Guaranteed Earnings for Delivery Workers

## Overview

SureShift is an AI-powered platform that protects delivery workers from income loss caused by external disruptions such as heavy rain, extreme heat, pollution, or floods.

Food delivery partners working with platforms like Swiggy and Zomato depend on daily deliveries to earn money. When disruptions happen, delivery orders decrease and workers lose their earnings.

SureShift helps solve this problem by predicting disruptions and automatically protecting workers’ income.

---

## Problem

Delivery workers earn money based on deliveries. When conditions like bad weather or environmental disruptions occur, delivery orders drop and workers lose income.

Currently, there is no simple system that protects gig workers from this type of income loss.

---

## Solution

SureShift is an AI-based parametric insurance platform that protects delivery workers’ earnings.

The system predicts disruption risks using weather and environmental data. Workers can activate protected shifts with guaranteed earnings. If deliveries decrease due to disruptions, the platform automatically verifies the situation and compensates the worker.

---

## Key Features

**AI Risk Assessment**
Predicts disruption risk using weather and environmental data.

**Protected Shifts**
Workers can activate protected working hours with guaranteed minimum earnings.

**Smart Risk Map**
Shows safe and risky delivery zones so workers can choose better areas.

**Automated Claims**
When disruption conditions are detected, claims are automatically triggered.

**Instant Payout**
Workers receive compensation instantly through UPI.

**Fraud Detection**
Claims are verified using GPS location, weather data, and delivery activity.

---

## Weekly Pricing Model

Basic Plan – ₹20 per week
Standard Plan – ₹35 per week
Premium Plan – ₹50 per week

The weekly pricing matches the typical earning cycle of gig workers.

---

## Parametric Triggers

Claims are triggered automatically when disruption events occur, such as:

* Heavy rainfall
* Extreme heat
* High pollution levels
* Flood alerts

---

## Technology Stack

Frontend: React.js
Backend: Node.js (Express)
Database: MongoDB
AI Services: Python (risk prediction and fraud detection)

APIs used:

* Weather API
* Air Quality API
* Map services (Google Maps / Mapbox)

Payments:

* UPI simulation or Razorpay test mode

---

## Workflow

1. Worker registers in the SureShift mobile app.
2. Worker selects a weekly plan or activates a protected shift.
3. The system monitors weather and disruption data.
4. If a disruption occurs, the system verifies the worker’s location and activity.
5. The claim is automatically approved and the payout is sent.
